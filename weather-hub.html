Dynamic Weather Hub

JAVASCRIPT (ES6) / FETCH API / CSS3

The Project

A sleek dashboard that fetches live weather data for any city in the world. It features a dynamic UI that changes background colors based on the temperature.

Core Features

• Real-time data fetching using the Fetch API.
• Error handling for invalid city names.
• Implementation of LocalStorage to remember the user's last searched city.

The Growth Moment

Dealing with Asynchronous JavaScript (Promises and Async/Await) was a huge milestone for me during this build. It taught me how to handle "waiting" for data without freezing the user interface.
