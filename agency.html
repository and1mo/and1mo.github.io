Creative Agency Page

HTML5 / CSS GRID / FLEXBOX

Design Focus

This project was an exercise in "pixel-perfect" design. I focused on modern layouts, clean typography, and mobile-first responsiveness.

Key Techniques

• CSS Flexbox: Used for navigation and alignment.
• CSS Grid: Used for the complex services section layout.
• Media Queries: Ensuring the site looks perfect from an iPhone to a 4K monitor.

What I Learned

I learned how to structure semantic HTML to improve accessibility and SEO, as well as how to use CSS variables to manage a consistent color palette across a whole site.
