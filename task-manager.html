Secure Task Manager

PHP / MYSQL / BOOTSTRAP

The Challenge

I wanted to build a tool that handled user data securely. The goal was to create a productivity app where users could register, log in, and save their tasks to a permanent database.

Technical Implementation

  • Backend: Built with PHP to handle server-side logic and session management.
  • Database: Designed a MySQL schema to store users and tasks with relational links.
  • Security: Implemented password hashing using `password_hash()` and protected against SQL injection using Prepared Statements.

Key Learning

This project taught me the importance of the Request-Response cycle and how to manage state across different pages using PHP Sessions.
